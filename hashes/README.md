# Hash Files Directory

This directory contains hash files for lab exercises.

## Usage
Place your hash files here:
- `md5_hashes.txt` - MD5 hashes
- `sha1_hashes.txt` - SHA1 hashes  
- `sha256_hashes.txt` - SHA256 hashes

## Generate Hashes
Run the hash generator script:
```bash
cd ../scripts
python3 generate_hashes.py
```

**Note:** Hash files are not committed to git for security reasons.
